description: PRs carry runtime evidence — verified in a real browser, with a GIF or screenshots
type: standard
author: hrithik
date: 2026-06-12
---
Static review and green checks are not proof a feature works: run it against a local server in a real browser and attach visual evidence — a GIF or screenshots of the flow — to the PR so reviewers see it working without checking out the branch. Auth-sensitive flows get verified against the deployed target, not only previews.
