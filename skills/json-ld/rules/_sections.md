# Rule Categories

Rules are prefixed by category:

- `org-` — Organization, Person, Service (business identity)
- `content-` — Article, BlogPosting, JobPosting, Course, Recipe, Event (content types)
- `nav-` — WebSite, BreadcrumbList, SiteNavigationElement (site structure)
- `ecom-` — Product (e-commerce)
- `interactive-` — FAQPage, HowTo (interactive rich results)
- `media-` — VideoObject (media types)
- `local-` — LocalBusiness (local SEO)
